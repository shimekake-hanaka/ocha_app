#### 3.イベント定義

| No. | 項目名               | イベント | 処理内容                   | 備考 |
| --- | -------------------- | -------- | -------------------------- | ---- |
| 1   | 日記作成ボタン         | クリック | 日記新規作成画面に遷移         |      |
| 2   | 日記詳細ボタン | クリック | [日記詳細画面](/admin/daily_post_show.md)（プレビュー画面）に遷移 |      |
| 3   | ページネーションボタン     | クリック | クリックしたページ番号に遷移     |      |
